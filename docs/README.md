# Documentation

This directory contains the documentation for the WireViz YAML Generator.

## Prerequisites

*   [Quarto](https://quarto.org/docs/get-started/) must be installed.

## Building the Documentation

To build the full website locally:

```bash
quarto render
```

This will generate the static site in the `_site` directory.

## Previewing

To preview the documentation with live reload:

```bash
quarto preview
```
